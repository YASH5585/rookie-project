# Product Roadmap

## MVP

- Local JSON profile system.
- Portfolio surface.
- Profile CMS.
- Resume generator.
- Cover letter generator.
- SOP generator.
- Professional bio generator.
- Readiness dashboard.
- Transparent benchmarking.
- GitHub and multi-platform coding analytics views from local data.
- TXT, PDF, and DOCX export.

## Next

- Real GitHub sync.
- Coding platform import flows for Codeforces, LeetCode, CodeChef, AtCoder, HackerRank, and GeeksforGeeks.
- More resume templates.
- Application tracker.
- Template gallery.
- Unit tests for scoring and generators.
- Lighthouse and accessibility CI.

## AI Layer

- Resume review.
- Cover letter personalization.
- Interview question generation.
- Project case study writer.
- LinkedIn summary generator.
- Career coaching prompts with transparent assumptions.

## SaaS Layer

- Auth.
- Cloud profiles.
- Billing.
- Premium templates.
- Team or campus dashboards.
- Versioned profile history.
